# starvax-player
Live show stage manager

Starvax player was born to have control over live stage show, by controlling music, videos and lights.
It is built on QT6 and FFMPEG, and interfaces with DMX protocol via USB.

Features:
- double media player with independent tracks;
- Script viewer with automatic scroll;
- DMX interface for stage lights control;
- OpenWebNet interface for hall lights;
- Integrated language for automating music and light changes;
- Network interaface for remote control;

This software is ever-under-developement but it has been used by some of my friends for over a decade.
